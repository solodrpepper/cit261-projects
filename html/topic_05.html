How to Use This Page

Enter in a "key : value" pair and then inert the data. Both the local and session storage sections will populate. Close the tab and then go back to the assignment page (effectively ending the session). By doing this, the session data will be completely erased. The Local storage persists. When you're finished, be sure to "Clear Storage". Also, don't enter in ANY sensitive information!
