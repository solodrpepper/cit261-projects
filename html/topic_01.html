For Loop

The "For Loop" is probably one of the most common loops used in programming. It's typically used when we know exatly how many time we need to run the loop.

While Loop

The while loop is great for when we don't know how many times we need to the loop to run. With this uncertainty, we still make sure to add a conditional statement to insure that it doesn't run forever.

Go ahead and input a starting number and end condition, or use the defaults and click to watch



Put in an end condition to see it run. Notice that even when zero is used, it will still run at least once.




Do Loop

The "Do Loop", or the "Do/While Loop" is similar to the "While Loop" except for the fact that a "Do Loop" will always run at least one time.

For Of Loop

The "For Of" loop it capable of looping through any object that is iterable. That includes arrays, maps, strings, sets, and any other user defined iterable objects! Cool, huh?
